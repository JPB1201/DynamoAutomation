##Instructions
- You'll need Dynamo 1.2.x or later for this to work
- You'll need to install the DynamoAutomation package
- Copy the contents of this folder to C:\temp\DynamoAutomation
- Run master graphs from DynamoSandbox.exe
- Reports, journal files and modified models will be written to C:\temp\DynamoAutomation\output
- If you're working with Revit 2017 you should upgrade the sample models first
